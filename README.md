WYMAGANE DO DZIAŁANIA:
1. przejdź do pliku test_ingTest.py
2. włącz terminal
3. wpisz komendy do instalacji:

        pip install pytest playwright

        playwright install


Uruchomienie testu:

1. przejdź do pliku test_ingTest.py
2. włącz terminal
3. wpisz "pytest -s" i kliknij Enter


Pipeline dla GitLaba jest w pliku .gitlab-ci.yml
